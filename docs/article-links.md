# Article Links

This file centralizes the main links associated with the study and its companion artifacts.

## Main supplementary repository

This repository provides the supplementary architectural, methodological, and documentation artifacts associated with the paper:

- `https://github.com/juniorvfj/stride-electoral-results-transmission`

## Proof-of-concept repository

The proof of concept (POC) used to support the empirical evaluation is maintained separately in a dedicated repository:

- `https://github.com/reinermaia/poc-stride-electoral/tree/main`

This separation was intentional. The present repository focuses on architecture artifacts, methodological documentation, bibliographic workflow, and supporting notes, while the external repository concentrates on the implementation and experimental artifacts of the POC.

## Relationship to the article

The repositories listed above are intended to complement the article by providing access to materials that could not be fully expanded in the paper due to page constraints, especially:

- high-resolution and editable architecture artifacts;
- exact search strings and source exports;
- intermediate corpus-processing files;
- enrichment and ranking outputs;
- POC implementation artifacts.

These materials should be understood as companion resources for transparency and inspection.
